Pokedex

Welcome, Trainer!

The Pokedex is a digital encyclopedia that catalogs every Pokemon species encountered on your journey — name, type, and a short description. Below is a handful of entries, rendered straight from a JavaScript array so you can see markup and data kept separate.

    SVG Demo: A Spinning Pokeball

    Every shape here (circle, path, rect) is a real DOM element you could inspect in dev tools, style with CSS, or animate with JS. This one animates itself using SVG's built-in <animateTransform>, no JavaScript involved.

    Canvas 2D Demo: Bouncing Shapes

    A handful of geometric shapes (circle, square, triangle) bounce around the canvas, each redrawn every animation frame using requestAnimationFrame.

    Canvas 3D-ish Demo: Rotating Wireframe Cube

    A cube defined by 8 corner points in 3D space, rotated every frame and projected onto the flat 2D canvas below — the same manual technique used before 3D APIs like WebGL existed.
